# 经济动态追踪 — 工作流约定

## 目录结构

```
reports/economy/         ← 本目录
├── CLAUDE.md            ← 本文件（工作流规则）
├── weekly/              # 周报（按自然周 周一~周日 一个文件）
└── daily/               # 日报（每天一个独立文件）
```

> 同级领域：`reports/ai/`、`reports/crypto/`、`reports/politics/` 等。

### "更新一下" 指令执行流程

当用户说 **"更新一下"** 时，执行以下步骤：

1. **调研当日经济新闻**
   - 使用 Exa / WebSearch 等工具搜索当天动态
   - 中英文源并行搜索，确保覆盖面
   - 搜索关键词包括但不限于：economy news today, Fed, GDP, inflation, 经济新闻, 央行, 就业, 贸易, 大宗商品 等

2. **更新周报文件**（追加模式）
   - 定位当前自然周对应的 `weekly/YYYY-MM-DD-to-MM-DD.md`
   - 如果该周文件不存在，创建新文件（参考已有周报格式）
   - 在日期条目区域**按日期顺序追加**当天内容
   - **"本周关键概念"表格**：追加当天新概念
   - **"本周趋势总结"**：仅在周日/周末最后一次更新时补充，平时保留占位符
   - **更新顶部"本周速览"表格**：追加当天摘要行

3. **创建日报文件**
   - 在 `daily/` 下创建 `YYYY-MM-DD.md`
   - 日报是独立完整的报告，包含详细分析（"为什么重要"段落）
   - 日报比周报条目更详细

4. **链接维护**
   - 周报之间通过顶部 `上一周` / `下一周` 互链
   - 日报顶部 `上一期` 指向前一天日报，`周报` 指向所属周报
   - 日报底部标注 `下一期日报：YYYY-MM-DD`

### 内容编辑规范

- **语言**：正文中文（简体），技术术语保留英文
- **星期标注**：务必核实日历，使用正确的星期几（如 2026-01-01 = 周四）
- **热度分级**：🔥 = 重大事件（5星），🏷️ = 值得关注（3-4星）
- **概念表**：每个新出现的经济术语/概念都要收录到概念表，附简明解释
- **来源标注**：每条新闻附带 `> 🔗` 来源链接
- **结构完整性**：周报中日期条目按时间顺序连续排列，概念表和趋势总结**始终放在文件底部**，不得插入日期条目之间
- **市场快照**：周报和日报均包含关键经济指标快照表格（股指、国债收益率、黄金、原油、汇率等）
