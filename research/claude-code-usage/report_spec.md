# Claude Code 使用方式与高效技巧 - 报告规格

## 报告元信息
- **标题**: Claude Code 使用方式与高效技巧：从入门到精通
- **受众**: Claude Code 用户（从入门到进阶）
- **目的**: 系统掌握 Claude Code 的高效使用方法和最佳实践
- **语言**: 简体中文
- **输出格式**: Markdown

## 范围定义
- **主题范围**:
  - 安装配置与环境准备
  - 核心功能详解与实战
  - 高级特性 (Skills, MCP, Hooks)
  - 工作流优化与效率提升
  - 常见问题解决
  - 最佳实践与进阶技巧

- **时间范围**: 2024-2026（关注最新版本特性）
- **地域**: 全球（中文输出）
- **深度**: 从基础操作到高级应用,覆盖完整学习路径

## 引用格式
- 行内引用格式: `[来源名称]`
- 每个关键声明都需要引用支持
- 数字和统计数据必须引用
- 最佳实践需要引用官方文档或权威来源

## 长度目标
- 总体篇幅: 10,000-15,000 字
- 各章节比例如上述百分比
- 包含充足的代码示例和配置示例
- 实战案例穿插其中

## 语气与风格
- **专业但易懂**: 技术准确但避免过度术语化
- **实用导向**: 每个概念都配合实际应用场景
- **结构化呈现**: 使用列表、表格、代码块清晰组织
- **渐进式学习**: 从简单到复杂,循序渐进
- **注释语言**: 中文为主,代码和技术术语保持英文

## 必须包含的来源类型
- ✅ 官方文档 (code.claude.com)
- ✅ Anthropic 官方博客
- ✅ GitHub 开源项目
- ✅ 社区最佳实践
- ✅ 实战教程与案例
- ✅ 专家经验分享

## 排除主题
- ❌ Claude API 底层技术细节
- ❌ 商业定价策略分析
- ❌ 与其他工具的详细对比评测
- ❌ 过时的配置方法

## 质量标准
1. **准确性**: 所有配置和命令需验证有效
2. **完整性**: 覆盖从入门到高级的完整路径
3. **实用性**: 每个技巧都可实际应用
4. **时效性**: 基于最新版本特性
5. **可操作性**: 提供清晰的步骤指引

## 特殊要求
- 代码示例使用 ```bash 或 ```json 等语法高亮
- 配置文件提供完整可用的示例
- 每个高级特性至少包含一个实战案例
- 问题解决部分提供明确的排查步骤
- 关键概念使用**加粗**或`代码格式`突出显示
