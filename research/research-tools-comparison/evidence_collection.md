## 关键发现汇总

### Claude Code Skills (S1, S2)
- **deep-research**: 9步流程，多轮并行起草+UNION合并，严格格式控制
- **tech-research**: 针对技术调研，多源搜索工具矩阵，活跃度筛选机制

### AI 原生调研工具
1. **NotebookLM** (S6, S14, S15): 文档交互式问答，最受研究者欢迎
2. **Elicit** (S4, S6, S7): 500万用户，AI科学研究助手
3. **Consensus** (S6, S7): 答案验证和相关研究发现
4. **Perplexity** (S12): 带引用的答案模式，学术深度搜索
5. **Scite** (S16, S17, S18): 引用上下文分析和可信度评估
6. **ResearchRabbit** (S13, S16, S17): 免费，文献映射和可视化
7. **Litmaps** (S13, S17, S18): 动态交互式文献地图
8. **Connected Papers** (S13, S17, S19): 引用关系图谱可视化
9. **Inciteful** (S17, S19): 高级引用关系可视化

### MCP 调研能力 (S5, S20, S21)
- Context7: 实时文档查询
- Perplexity AI MCP: 研究助手式交互
- deepwiki: GitHub 项目深度文档
- exa: 网络搜索和代码上下文
- Sequential Thinking: 结构化问题解决

### 传统方法论
- **PRISMA 2020** (S3, S9): 系统性综述金标准，27项清单，2024多个扩展更新中
- **扎根理论** (S25, S26, S27): 从数据归纳理论，质性研究核心方法
- **内容分析法** (S27): 系统化文本分析
- **系统调查法** (S23): 多维数据整合

### 文献管理工具 (S28, S29)
- **Zotero**: 开源免费，价值领先
- **Mendeley**: 协作笔记出色，2024取消扩展机构许可
- **EndNote**: 企业级，高度定制，最多1000人协作

## 工具分类框架

### 按自动化程度
1. **全自动化**: Claude Code skills (deep-research, tech-research)
2. **AI 辅助**: NotebookLM, Elicit, Consensus, Scite
3. **半自动化**: ResearchRabbit, Litmaps, Connected Papers
4. **手动+工具**: Zotero, Mendeley, PRISMA

### 按研究类型
1. **技术调研**: tech-research skill, Context7 MCP, deepwiki MCP
2. **学术综述**: deep-research skill, Elicit, Scite, PRISMA
3. **文献映射**: ResearchRabbit, Litmaps, Connected Papers
4. **多文档问答**: NotebookLM, Paperguide
5. **引用管理**: Zotero, Mendeley, EndNote

### 按证据质量控制
1. **严格控制**: PRISMA, deep-research skill (证据表+引用验证)
2. **中度控制**: Scite (引用上下文), Elicit (来源标注)
3. **基础控制**: NotebookLM, Consensus
4. **用户主导**: Zotero, ResearchRabbit
